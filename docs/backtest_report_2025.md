## 戦略概要

### 使用モデル
- **アルゴリズム**: LightGBM + CatBoost のアンサンブル（Meta Stacking）
- **学習方式**: Walk-Forward（月次ローリング再学習）
- **特徴量数**: **213個**（血統、過去戦績、騎手・調教師統計、コース適性など）

### 投資戦略

| パラメータ | 設定値 | 説明 |
| :--- | :--- | :--- |
| **Kelly係数** | 0.05 (5%) | 資金の5%を上限として賭ける |
| **EV閾値** | 1.0 | 期待値が1.0以上のベットのみ実行 |
| **1レース上限** | 5,000円 | リスク管理のための上限 |
| **初期資金** | 100,000円 | シミュレーション開始時の資金 |
| **対象券種** | 単勝・複勝・馬連 | 3連単・3連複は含まず |

## 検証結果

### データリーク検証（合格）
- **Market Sanity Test**: ROI 88.5%（正常値）
- **Placebo Test**: ROI 62.9%（予測無効化で損失発生→正常）
- **結論**: システムとしての致命的なリークは解消済み
