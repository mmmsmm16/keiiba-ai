# Discord Webhook 設定ガイド

Keiiba-AIの通知を受け取るための「Discord Webhook URL」を取得する手順です。
Discordのアカウントをお持ちでない場合は、先にアカウント作成をお願いします。

## 手順 1: サーバーの作成（または既存サーバーの選択）
通知を受け取りたいサーバー（グループ）を用意します。自分ひとりだけのサーバーを作成するのがおすすめです。

1. Discord左側のサーバーリストの一番下にある **「＋」（サーバーを追加）** をクリック。
2. **「オリジナルの作成」** → **「自分と友達のため」** を選択。
3. サーバーに適当な名前（例: `Keiiba-AI Notifications`）をつけて **「新規作成」**。

## 手順 2: 通知用チャンネルの作成
デフォルトの `#general` でも良いですが、専用チャンネルを作ると管理しやすいです。

1. サーバー内のチャンネルリストの「＋」を押して、テキストチャンネルを作成（例: `#race-alerts`）。

## 手順 3: Webhook の作成
ここが重要です。

1. 通知を送りたいチャンネル（例: `#race-alerts`）の横にある **歯車アイコン（チャンネルの編集）** をクリック。
2. 左側のメニューから **「連携サービス (Integrations)」** をクリック。
3. **「ウェブフック (Webhooks)」** のパネルをクリック（または「ウェブフックを作成」ボタン）。
4. **「新しいウェブフック」** ボタンをクリック。
5. 作成されたウェブフック（「Spidey Bot」などの名前になっています）をクリックして設定を開く。
    - **お名前**: 好きな名前に変更（例: `AI予想家`）。
    - **チャンネル**: 正しいチャンネルになっているか確認。
6. **「ウェブフックURLをコピー (Copy Webhook URL)」** ボタンをクリック。

## 手順 4: 環境変数への設定
コピーしたURLを `Keiiba-AI` システムに設定します。

1. プロジェクトフォルダ (`c:\Users\masat\MyLab\Dev\keiiba-ai`) にある `.env` ファイルを開く（なければ作成）。
2. 以下の行を追加（または書き換え）します：

```env
DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/1234567890/AbCdEfGhIjKlMnOpQrStMw...
```

※ `https://...` の部分は、先ほどコピーした実際のURLを貼り付けてください。

---
以上で設定は完了です！
`src/scripts/auto_predict.py` が実行されると、このチャンネルに通知が届くようになります。
