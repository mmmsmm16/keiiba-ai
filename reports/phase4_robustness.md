# Phase 4: Backtest & Robustness Analysis Strategy Report

**Date**: 2025-12-15
**Model**: v12 (v11 Features - Human Stats)
**Protocol**: Walk-Forward Validation (2021-2023) with Slippage Simulation

## 1. Executive Summary

Phase 3で確定した**モデルv12**（Human Stats除外版）の堅牢性（Robustness）検証を実施した。
スリッページ（オッズ低下）環境下でのROI感度を分析した結果、**スリッページ係数1.0（定価）においてもROIは平均69.18% (Kelly)** であり、損益分岐点（100%）には到達していないことが確認された。
しかし、Naive（単勝1番人気買い）のROI 47.35%に対し、**+21.8%の改善**を実現しており、モデルの予測能力自体は高い。

## 2. Sensitivity Analysis (Mean over 2021-2023)

オッズが不利になった（Slippage）場合のROI変化：

| Slippage Factor | ROI (Naive) | ROI (Kelly) | ROI (EV Flat) | Memo |
| :--- | :--- | :--- | :--- | :--- |
| **1.00 (None)** | 47.35% | **69.18%** | 67.84% | Baseline Recovery Rate |
| 0.95 (5% Loss) | 44.98% | 65.40% | 63.86% | |
| 0.90 (10% Loss) | 42.61% | 61.57% | 60.07% | |
| 0.85 (15% Loss) | 40.25% | 58.17% | 56.41% | |

*   **感度**: オッズが5%悪化すると、Kelly ROIは約3.8ポイント低下する。線形に近い挙動であり、特定の閾値で崩壊するような不安定さは見られない。

## 3. Discussion: Why ROI < 100%?

1.  **単勝の限界**: 現在のシミュレーションは「単勝（Win）」のみを対象としている。
    *   単勝の控除率（テラ銭）は約20-25%であり、期待値回収率は75-80%が目安。
    *   Naive ROIが47%と極端に低いのは、モデルが「過剰人気馬（オッズ妙味のない馬）」を高確率で推奨しているか、検証データのオッズ分布（特に1番人気）の平均配当が低い可能性がある。
2.  **Kelly戦略の有効性**:
    *   Naive (47%) -> Kelly (69%) の大幅な改善は、**「勝つ確率が高いがオッズが見合わない馬」への賭け金を減らし、「期待値が高い穴馬」を拾えている** ことを示唆する。
3.  **Next Step**:
    *   **馬券種の拡大**: 単勝では限界があるため、馬連・三連複など高配当・高分散な券種への展開が必要。
    *   **EV閾値の厳格化**: 現在は `EV > 0` で賭けているが、`EV > 1.2` など厳選することで的中率は下がるがROIは向上する可能性がある。

## 4. Conclusion

*   **Robustness**: モデルは安定しており、特徴量削除（v12）は成功している。
*   **Profitability**: 現状の単勝戦略では黒字化は困難。Phase 5以降で戦略（券種・資金配分）の抜本的な高度化が必要である。
